## ğŸ› ï¸ Getting Started

### Prerequisites

- Node.js (v16+ recommended)
- npm

### Installation

1. **Clone the repository:**
   ```bash
   git clone <repo-url>
   cd SkillBridge
   ```

2. **Install dependencies for both backend and frontend:**
   ```bash
   cd backend
   npm install
   cd ../frontend
   npm install
   ```
